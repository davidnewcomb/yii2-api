Yii 2 Forum Engine Module API

## Installation

Run console command

```
composer require yii-podium/yii2-api
```

Or add the package to your `composer.json` file:

```
{
    "require": {
        "yii-podium/yii2-api": "^0.1"
    }
}
```

then run `composer update`.
